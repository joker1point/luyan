“未来商业AI挑战赛”详情-参赛选手版

## **一、大赛总览**

###  1. 大赛名称

未来商业AI挑战赛

###  2. 大赛主题

启动你的第一次AI创业

###  3. 大赛口号

用AI构建可持续的商业未来

###  4. 大赛背景与核心目标

（1）大赛背景

当前AI技术应用已进入爆发临界点，行业发展的核心瓶颈并非技术本身，而是能够深刻洞察应用场景、善于用技术构建可持续商业模式的创新型人才。高校作为人才培育的核心阵地，蕴藏着最具活力、最具前沿视野的AI人才与创意灵感。

本届“未来商业AI挑战赛”搭建起一扇“开放式创新”之门——这不仅是一场竞技赛事，更是一次真实的商业邀约。面向顶尖学生团队发出诚挚邀请，将自身的AI创意，锤炼成可落地、可增长、可闭环的商业方案，依托产业资源与资本视野，验证商业构想、开启AI职业与创业生涯。

（2）核心目标

创新发掘目标：选拔并孵化一批具备清晰商业模式、可落地产品原型的学生AI创业项目，推动项目达到可投资、可产业试点的水平，助力AI创意从“想法”走向“实践”。

人才引进目标：搭建AI技术、创意灵感与市场需求的对接桥梁，发掘一批兼具技术能力、商业思维与创新潜力的优秀团队与个人，为行业储备优质人才。

###  5.大赛核心定位

本次大赛核心定位为“AI创业大赛”，区别于单纯的技术竞赛或应用展示赛。参赛者需基于大赛指定赛道方向，完成一款可交互、可演示、具备清晰应用场景的产品原型（形式不限，可采用Web、APP、小程序等），而非仅提交算法模型或创意方案。大赛核心宗旨是发现、验证并孵化具备完整商业模式、快速增长潜力的早期AI创业项目，助力学生实现“第一次AI创业”的落地尝试。

###  6. 大赛组织架构

主办单位：广州霆万科技有限公司、广东工业大学计算机学院团委

协办单位：“匠心创梦”创新创业辅导员工作室、广东工业大学计算机学院学生会、研究生会

## **二、参赛对象与产品要求**

###  1. 参赛对象

广东工业大学在读学生（本科/研究生/博士），不限专业，鼓励跨专业、跨院系组队参赛。

###  2. 组队要求

 可单人参赛，也可自由组队，每队人数最多不超过3人。

 允许跨年级、跨专业组队，组队后不得随意更换队员。

 同一参赛队员仅可报名加入一个队伍，若发现重复报名，将取消其所有参赛资格。

###  3. 产品要求

 本次大赛核心挖掘具有实际落地潜力的AI应用项目，参赛团队需完成一款AI应用工具（形式不限，可采用Web、App、小程序等），具体要求如下：

（1）核心技术要求

产品须显著运用AI相关技术，具备处理或生成文字、图片、音频、视频中至少一种内容的核心能力，AI技术需与产品场景深度结合，避免“技术堆砌”。

（2）交付标准

产品需具备清晰的应用场景、明确的目标用户，提交可直接演示的产品原型或Demo（可提供演示视频、在线试用链接、安装包等形式），确保评委可直观体验产品核心功能。

**（3）赛道设置**

本次大赛设置两大赛道，参赛团队可任选其一报名，不可跨赛道参赛：

**赛道一：自由创想赛道**

不限行业、场景，鼓励参赛团队从校园生活、个人兴趣、社会需求或未来发展趋势出发，通过AI技术创造全新产品形态，大胆探索AI的创新应用边界。

**赛道二：行业命题赛道（可选）**

参赛团队需在限定方向内，交付高完成度的AI应用工具，聚焦具体行业场景，提升产品的落地可行性，参考方向如下：

参考方向1：学习场景AI工具——聚焦高效学习、精准辅导、心灵陪伴、知识管理等细分场景，解决学生学习和生活痛点。

参考方向2：营销场景AI工具——一站式生成营销文案、社媒图片、短视频及配音，助力中小商户、创作者降低营销成本。

参考方向3：文化创意AI工具——包括AI绘本生成器（文生图+旁白合成）、交互式音乐教学App（音律分析+可视化指导）等。

参考方向4：视频处理AI工具——在录屏、观看视频过程中，实现实时语音转文字，并自动抓取视频关键帧，生成可跳转、可搜索的电子版资料，提升信息获取效率。

参考方向5：行业服务AI工具——为法律、电商、设计、教育等行业，提供智能文档处理、素材生成、流程优化等相关工具。

##  **三、评审机制与标准**

###  1. 评审委员会构成

评审委员会由广州霆万科技有限公司CEO、产品专家、营销专家和广东工业大学技术专家组成，确保评审过程的专业性、公正性、权威性，兼顾技术实力与商业潜力的双重考量。

###  2. 核心评审标准（总分100分）

总评分=创新性（25分）+技术实现（25分）+用户体验与设计（20分）+应用潜力（20分）+现场表现（10分）

| **评审维度**   | **分值** | **具体考察点**                                               |
| -------------- | -------- | ------------------------------------------------------------ |
| 创新性         | 25分     | 产品理念、应用场景或技术实现的独特性、前瞻性，是否能解决真实痛点，区别于现有同类产品。 |
| 技术实现       | 25分     | AI技术运用的合理性、技术复杂度，产品原型的稳定性、完成度，核心功能的可演示性。 |
| 用户体验与设计 | 20分     | 产品交互逻辑的简洁性、界面设计的美观度，多模态交互的自然流畅程度，是否符合目标用户使用习惯。 |
| 应用潜力       | 20分     | 市场需求的明确性、商业前景或社会价值的可行性，商业模式的完整性，规模化发展的潜力。 |
| 现场表现       | 10分     | 方案的完整性与逻辑性，路演表达的清晰度、说服力，答辩过程中的应变能力及团队协作表现。 |

若多支队伍总分相同，按以下优先级排序：创新性维度得分高者排名靠前；若创新性得分仍相同，技术实现维度得分高者排名靠前；若以上两项均相同，由评审委员会集体投票决定最终排名。
