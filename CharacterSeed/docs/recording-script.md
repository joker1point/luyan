# CharacterSeed 路演录屏拍摄脚本

> **配套文件**：`scripts/record_pitch.py` + `docs/pitch-deck.pptx` + `docs/demo-script.md`
> **总时长**：6'10"（路演时长） + 拍摄准备 5 分钟
> **输出**：`docs/pitch-recording-{时间戳}.mp4` (1920×1080, 30fps, H.264+AAC)
> **设计日期**：2026-06-27

---

## 🎬 拍摄前 5 分钟（一次性准备）

| 步骤 | 操作 | 时长 |
|---|---|---|
| 1 | 关闭无关软件（钉钉/微信/邮件/浏览器其他 tab） | 1 min |
| 2 | 设置电源模式为"高性能"（避免自动休眠） | 10 s |
| 3 | 关闭 Windows 通知焦点助手 + 勿扰模式 | 10 s |
| 4 | 测试麦克风：`python scripts/record_pitch.py --list-devices` 找到设备名 | 30 s |
| 5 | PowerPoint 打开 `pitch-deck.pptx` → 进入排练计时模式（可选） | 1 min |
| 6 | 录制端电脑 / PowerPoint 双屏准备（推荐 PPT 切到演示者视图） | 1 min |
| 7 | 录制前 30 秒喝口水、深呼吸 | 30 s |

## 🎬 录制后 5 分钟（剪辑 + 压缩）

### Step 1：检查文件
```powershell
Get-ChildItem docs/*.mp4 | Select-Object Name, Length, LastWriteTime
```

### Step 2：剪辑（去掉开头的"3、2、1"准备段）
```powershell
# 假设录制总时长 6'20"，前面有 5 秒准备，从第 5 秒开始
ffmpeg -i docs/pitch-recording-20260627-153000.mp4 -ss 5 -c:v libx264 -crf 20 -c:a aac final-pitch.mp4
```

### Step 3：压缩（如果文件 > 100 MB）
```powershell
ffmpeg -i final-pitch.mp4 -c:v libx264 -crf 23 -preset slow -c:a aac -b:a 128k pitch-compressed.mp4
```

### Step 4：添加字幕（可选，用 ffmpeg + 字幕文件）
```powershell
ffmpeg -i pitch-compressed.mp4 -vf "subtitles=pitch-subs.srt" pitch-with-subs.mp4
```

## ⚠️ 风险预案

| 风险 | 应对方案 |
|---|---|
| 麦克风没声 | 录制前 1 分钟先用 Windows 录音机测试 |
| 录制中网络卡顿 | 全程**不联网**录制（PPT + Demo 都本地） |
| Demo 角色答非所问 | 提前 3 次预演，固定 prompt 模板；翻车就切 P8 商业版图 |
| 录制中来电 | 手机静音 / 飞行模式 |
| 录制超时（> 7 分钟） | 直接在 P9 切 CTA，砍 P10 致谢段 |
| 录制文件 > 500 MB | 录制时 `--preset ultrafast --crf 23`，录完用上面 Step 3 压缩 |
| 录出来是黑屏 | 检查 PowerPoint 是否在"幻灯片放映"模式；或用 OBS 替代 |

## 💡 进阶优化

### 1. 用 OBS 替代 ffmpeg（界面化、更稳）
- 下载：https://obsproject.com/
- 设置：来源 = "显示器捕获" + "音频输入捕获"
- 输出：设置 → 输出 → 录像 → 编码器 H.264 / 速率控制 CBR / 比特率 8000 Kbps

### 2. 提词器（让眼睛看镜头）
- PowerPoint 演示者视图自带提词器
- 视图 → 演讲者备注 → 勾选"使用演讲者视图"
- 把这份脚本的"口播"内容复制到每页 PowerPoint 备注

### 3. 自动播放（无人值守录制）
- PowerPoint → 幻灯片放映 → 设置幻灯片放映
- 勾选"使用排练计时"
- 给每页设置"切换" → "之后：xx 秒"（按本脚本节拍）

### 4. 字幕自动生成（whisper）
```bash
# 安装 openai-whisper
pip install openai-whisper

# 用 base 模型（最快）或 medium 模型（更准）
whisper final-pitch.mp4 --language zh --model base --output_format srt
```
